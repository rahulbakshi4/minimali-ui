List

The list component is a continuous group of text, images and icons that may contain primary or supplemental actions.

Basic List

Basic List can be implemented using .list-group, .list-title, .list-item classes.

The default list come with no list style but disc and decimal list style can be added using .list-disc and .list-decimal classes.
